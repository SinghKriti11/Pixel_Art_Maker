# 🎨 Pixel Art Generator

An interactive **Pixel Art Generator** built using **HTML, CSS, and JavaScript** that allows users to create pixel-based drawings on a customizable grid.

---

## 🚀 Features

- 🎛️ Adjustable grid size (width & height)
- 🎨 Custom color picker
- ✏️ Paint mode for drawing
- 🧽 Erase mode
- 🗑️ Clear grid functionality
- 🌙 Dark mode toggle (saved using localStorage)
- 📱 Touch support (mobile-friendly)

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

---

## ⚙️ How to Use

1. Select grid width and height  
2. Click **Create Grid**  
3. Choose a color  
4. Use:
   - **Paint** → draw  
   - **Erase** → remove color  
5. Click **Clear Grid** to reset  
6. Toggle **Dark Mode** if needed 

---

## 📱 Responsive Design
1. Supports touch drawing option
2. Dark Mode
3. Toggle between light & dark theme (Preference saved using localStorage)
